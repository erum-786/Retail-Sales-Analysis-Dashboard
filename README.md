# 🛍️ Retail Sales Analysis Dashboard

An interactive Power BI dashboard designed to monitor, analyze, and visualize retail sales performance. This project transforms raw retail data into actionable business insights, helping stakeholders make data-driven decisions.

## 📊 Dashboard Overview
This project provides a comprehensive view of retail operation, representing:-
* **Total Sales:** Displays the aggregate monetary value of all transactions.
* **Total Customers:** Shows the distinct number of unique buyers.
* **Quantity Ordered:** Represents the total volume of products sold.

### 📈 Visualizations & Charts
* **Sum of Sales by Year:** Highlights annual revenue growth and trends.
* **Sum of Sales by Country:** Pinpoints regional sales performance and market reach.
* **Sum of Sales by Order Timing:** Visualizes sales trends by time of day to identify peak business hours.
* **Count of Invoice by Day Name:** Displays transaction frequency to understand daily customer activity.

### 🎛️ Interactive Filters (Slicers)
The dashboard is highly interactive, allowing users to slice data using the following filters:
* **Year:** Drill down into historical performance trends.
* **Month:** Represent data using months choice.
* **Country:** Isolate data for targeted regional analysis.

## 🛠️ Built With
* **[Microsoft Power BI]** - Data visualization and dashboard creation.
* **DAX** - Calculated measures used for KPI cards and custom metrics.
